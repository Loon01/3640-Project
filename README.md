# 3640-Project

======= Prerequisites: =========

- Python 3.x installed on your system.

- Need to install pygames, to install do:

  - python3 -m pip install -U pygame --user

- if any difficulty, refer to (https://www.pygame.org/wiki/GettingStarted)


======= Running file: =========

- To run file, do:

  - python3 snake.py
